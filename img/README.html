图片资源说明

需要添加的图片

将以下图片放在 source/img/ 目录下：

1. 网站图标（favicon）

  • 文件名: favicon.png 或 favicon.ico
  • 尺寸: 32x32 或 64x64 像素
  • 用途: 浏览器标签页图标

2. 头像

  • 文件名: avatar.jpg 或 avatar.png
  • 尺寸: 建议 200x200 像素
  • 用途: 侧边栏作者头像

3. 顶部背景图（可选）

  • 文件名: top_img.jpg
  • 尺寸: 1920x1080 像素（16:9 比例）
  • 用途: 首页顶部大图

4. 默认文章封面（可选）

  • 文件名: cover1.jpg, cover2.jpg, cover3.jpg 等
  • 尺寸: 建议 1200x630 像素
  • 用途: 文章卡片封面图

图片获取建议

  • 免费图片网站:

    • Unsplash: https://unsplash.com/
    • Pexels: https://www.pexels.com/
    • Pixabay: https://pixabay.com/

  • 图标生成:

    • Favicon Generator: https://realfavicongenerator.net/
    • Icon Archive: https://iconarchive.com/
